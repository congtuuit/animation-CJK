Animation CJK

Samples

    Description

    AnimCJK contains SVG files to draw Japanese kana or kanji and simplified or traditional Chinese hanzi stroke by stroke.

    The Japanese repository contains "Kana" (177 characters), "Jōyō kanji" (2136 characters), "Jinmeyō kanji" (863 characters) and some other characters ("Hyōgai kanji", "Components" and "Strokes").

    The Korean repository contains some "Hanja".

    The simplified Chinese repository contains "Commonly used hanzi" (7000 characters) and some other characters ("Uncommon hanzi", "Traditional hanzi used with simplified hanzi", "Components", "Bopomofo" and "Strokes").

    The traditional Chinese repository contains "HSK v3 level 1", "HSK v3 level 2", "HSK v3 level 3" and some other traditional hanzi.
